# Ethan Yates - Portfolio

## Project Description
This is a personal portfolio website for Ethan Yates, an aspiring software developer and IT professional.  
The site demonstrates advanced layout techniques using **CSS Grid and Flexbox**, responsive design, accessibility best practices, and a dark mode theme system.

The portfolio includes:

- Hero section with text and avatar
- Projects section (CSS Grid)
- Skills section (Flex/Grid)
- About section
- Footer with GitHub link

The site is **mobile-first** and includes hover transitions and a fade-in animation for polished UI.
